# TODO LIST 프로젝트

이 프로젝트는 간단한 할 일 관리 웹 애플리케이션입니다.

## 기능
- 할 일 추가하기
- 할 일 삭제하기
- 모든 할 일 지우기

## 사용한 기술
- HTML
- CSS
- JavaScript

## 웹사이트 링크
[TODO LIST 바로가기](https://vibe-coding-demo.vercel.app/)

## 프로젝트 실행 방법
1. 프로젝트를 클론합니다
```bash
git clone https://github.com/seokyoungjin/ViBE-CODING-DEMO.git
```
2. 프로젝트 폴더로 이동합니다
```bash
cd ViBE-CODING-DEMO
```
3. 웹 브라우저에서 index.html을 엽니다
